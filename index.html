Professional Work

Tax Appeal Plus — Property Tax Grievance Platform

I develop and maintain Tax Appeal Plus, an enterprise-level multi-tenant CRM and workflow automation platform used by high-volume property-tax-appeal firms across New York State and beyond. I designed and implemented the multi-tenant architecture, enabling the platform to serve multiple clients with isolated data, configurations, and workflows within a single application instance.

As a backend developer and API developer, I build RESTful APIs and Web APIs using ASP.NET Core and C# programming. I actively maintain two product versions in parallel—the original ASP.NET Web Forms 4.0 product and the newer ASP.NET Core MVC rebuild—while keeping feature parity. Both front-ends rely on a shared ASP.NET Core Web API project that I design, secure, and optimize with Entity Framework Core and SQL Server to enforce tenant isolation and consistent business rules.

The multi-tenant application handles client intake, document automation, document filing, invoicing, and reporting—consolidating operations that were traditionally scattered across spreadsheets, word-processors and multiple disconnected tools. My contributions enable property-tax grievance firms to scale efficiently, reduce manual errors, and deliver faster turnaround on thousands of appeals annually through improved software engineering practices, multi-tenant architecture design, and backend development expertise.
